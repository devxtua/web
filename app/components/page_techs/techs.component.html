Технологии

HTML5 - альфа и амега. Основные функции его это создание заголовков, выделений абзацев с помощью тегов. Сам по себе язык Html не является языком программирования, то есть не может создавать динамические функции вместо этого он способен организовывать и форматировать документы.
CSS3 - Chevrolet Chevelle SS. Это язык, который отвечает за визуальное представление документов пользователю. В современном интернете практически ни один сайт не обходится без CSS, поэтому HTML и CSS действуют в единой связке. Простыми словами это язык с помощью которого программисты добавляют (новые краски) сайту.
PHP - просто "слон". Инструмент для создания персональных веб-страниц, один из популярных языков программирования. Основные функции это автоматизация отправки HTTP-заголовков, работа с HTTP-авторизацией, обработка файлов загружаемых на сервер. Язык программирования PHP поддерживается почти на всех известных платформах. Иными словами PHP язык программирования созданный для общения с сервером.
JavaScript - "вездесущ и везде орущ". Мультипарадигменный язык программирования, который позволяет применять сложные вещи на web странице. С помощью него можно создать обновляемый контент, управляет мультимедиа, он может анимировать изображения, впрочем, делает всё, что угодно.

Фреймворки и CMS

jQuery - это набор функций JavaScript, фокусирующийся на взаимодействии JavaScript и HTML. JQuery не отдельный язык программирования и работает в сочетании с JavaScript. JQuery - это кроссплатформенная JavaScript-библиотека, предназначенная для упрощения клиентского сценария HTML.
Angular - "шоб ему пусто было", это открытая и свободная платформа для создания, веб-приложений, написанная на языке TypeScript. Angular был создан командой компании Google. То есть Angular является фреймворком от компании Google для создания веб-приложений.
WordPress - "пхни куда хош", это веб-приложение, которое позволяет владельцам сайтов, редакторам, авторам управлять их сайтами и публиковать контент безо всяких знаний программирования. WordPress пользуеться PHP и MySQL, потому что они поддерживаются почти всеми хостинг-провайдерами. Очень часто, эту CMS используют для создание блога.
OpenCart - это быстрая и нетребовательная к ресурсам сервера CMS для магазина. Это — очень надёжное и простое CMS, для желающих создать свой онлайн магазин при минимальных затратах. OpenCart построен по принципу MVC, поддерживает мульти-шоп. Относительно бюджетная реализация любого функционала, установкой дополнительных модулей или силами программиста. OpenCart пропагандирует безопасность.

Програмы учета и CRM-системы

Простой софт - это программный код платформы, скомпилированный в исполняемый файл и обеспечивающий работу любой конфигурации базы данных. Программа оказывает услуги по автоматизации бизнеса любого вида и масштаба и реализует ИТ-проекты.
1С - единая платформа для автоматизации деятельности организации: бухгалтерского, кадрового, управленческого и финансового учета. Все конфигурации построены на общих принципах, их можно настраивать и изменять под специфику организации.
Битрикс24 – это набор из пяти важных и полезных инструментов, которые помогают бизнесу работать: CRM, онлайн-офис, задачи и проекты, контакт-центр, сайты и магазины. С помощью него вы можете выставлять и выполнять задачи, создавать план рабочего времени и общаться с коллегами так же легко, как вы это делаете в социальной сети.
